
# Projeto: Plataforma de Previsão de Preços de Aluguel de Imóveis

---

## Objetivo:
Criar uma plataforma simples e eficiente para prever o preço de aluguel de imóveis em Lisboa, com base em características como tamanho, número de quartos e localização. A plataforma será funcional, rápida e com uma interface limpa e objetiva.

---

## Funcionalidades Principais:

1. **Previsão de Preço de Aluguel**:
   - O usuário insere as características do imóvel (tamanho, número de quartos, localização) e o sistema retorna o preço estimado de aluguel.
   
2. **Filtros Simples**:
   - Permitir que o usuário filtre imóveis por localização e faixa de preço.

3. **Exibição de Resultados**:
   - Exibição simples de resultados em formato de lista ou tabela.

---

## Arquitetura do Projeto

---

### 1. Coleta de Dados  
- **Fontes**: Web Scraping (Idealista, OLX, Imovirtual, etc.) ou API de Imóveis.
- **Processo**: Utilização de Python com **BeautifulSoup** ou **Selenium** para extrair dados como localização, tamanho e preço de aluguel dos imóveis.

---

### 2. Pré-processamento de Dados  
- **Limpeza de Dados**: Remoção de registros incompletos.
- **Transformação**: Ajustes nas variáveis de localização, tamanho e quartos para análise.
- **Normalização**: Normalização das variáveis contínuas (tamanho, número de quartos) para garantir boa performance do modelo.

---

### 3. Modelagem de Dados  
- **Modelo de Regressão Linear** para prever o preço de aluguel com base em dados históricos.
- **Tecnologias**: **Scikit-learn**, **Jupyter Notebooks** para protótipos rápidos.

---

### 4. Backend  
- **Tecnologias**: **Flask** para criar a API simples.
- **Endpoints**: 
   - `/predict`: Previsão do preço de aluguel.
   - `/filter`: Filtros simples de localização e preço.
   
---

### 5. Frontend  
- **Tecnologias**: **HTML/CSS/JavaScript** para interface simples e interativa.
- **Exibição**: Resultado da previsão mostrado de maneira clara em formato de texto ou tabela.

---

### 6. Deploy e Hospedagem  
- **Plataforma de Deploy**: **Heroku** ou **AWS Elastic Beanstalk** para hospedar o backend.
- **Banco de Dados**: **SQLite** ou **JSON**.

---

## Fluxo de Uso

1. O usuário preenche um formulário com o **tamanho (m²)**, **número de quartos**, **localização**.
2. Ao clicar em **“Prever Preço”**, o sistema retorna o preço de aluguel estimado.
3. Filtros de **localização** e **faixa de preço** são aplicados para exibição de resultados.
4. **Exibição** dos imóveis que atendem aos critérios.

---

## Conclusão
Este projeto é uma plataforma de **previsão de preços de aluguel** simplificada, com foco na **eficiência** e **rapidez**, utilizando técnicas de aprendizado supervisionado e uma interface mínima.
